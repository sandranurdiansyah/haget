🔗 **Live:** [sandranurdiansyah.github.io/haget](https://sandranurdiansyah.github.io/haget)

---

## Apa itu Haget?

Haget adalah tools gratis untuk siapapun yang ingin mulai freelance, masuk gig economy, atau buka usaha sendiri — tapi belum tahu harus mulai dari mana.

Cukup input skill dan alat kerja yang sudah kamu punya, Haget akan mencocokkan dengan 40+ peluang karir di berbagai sektor: digital, kuliner, konstruksi, kecantikan, pertanian, dan lainnya.

---

## Fitur

- 🎯 **Matchmaking berbasis skill & tools** — rekomendasi dihitung dari kecocokan nyata, bukan acak
- 🌱 **3 mode pencarian** — Pengalaman, Alat Kerja, atau Semangat Juang (random)
- 📊 **Dashboard & readiness score** — visualisasi kesiapan karir kamu
- 📸 **Upload foto profil** — crop otomatis 1:1, embed langsung ke CV
- 📄 **Export CV sebagai HTML** — standalone file, bisa dibuka offline & print ke PDF
- ⚡ **100% client-side** — tidak ada backend, tidak ada data yang dikirim ke server

---

## Tech Stack

```
Vanilla HTML + CSS + JavaScript
JetBrains Mono + Inter (Google Fonts)
Tabler Icons
EmailJS (notifikasi)
Canvas API (crop foto)
```

Tidak ada framework. Tidak ada dependency npm. Satu file HTML.
